## Getting Started
### See Live Site [HERE](https://brave-tree-0c566030f.3.azurestaticapps.net/)

### Installation
1. Fork and Clone the repo
   ```sh
   git clone https://github.com/madalinefitz/Two-or-More
   ```
2. Install npm packages
    ```sh
    npm install
    ```
4. Start the Front End
    ```sh
    npm start
    ```
